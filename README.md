# Blinkit Sales Analysis Dashboard

An interactive Power BI dashboard built to analyze Blinkit's sales performance, outlet operations, product distribution, and customer ratings through dynamic visualizations and business intelligence techniques.
Project Overview

This project showcases an interactive "Blinkit Sales Analysis Dashboard" developed in - "Power BI" to analyze sales performance, outlet operations, product distribution, 
and customer ratings. The dashboard enables users to explore key business metrics through dynamic visuals and interactive filters, helping uncover valuable insights for decision-making.

# Objectives :

1. Monitor overall sales performance.
2. Analyze sales by item type, outlet size, and outlet location.
3. Compare outlet performance across different establishment years.
4. Track customer ratings and product distribution.
5. Visualize sales geographically across India.
6. Build an interactive dashboard for business intelligence.

# Tools & Technologies

1. Power BI
2. Power Query
3. DAX
4. Microsoft Excel / CSV
5. Data Modeling

# Dashboard Features

1. KPI Cards (Total Sales, Total Orders, Total Outlets, Average Rating)
2. Sales by Item Type
3. Sales by Outlet Size
4. Sales by Outlet Location
5. Outlet Establishment Analysis
6. Average Rating by Item Type
7. India Sales Map
8. Interactive Slicers and Filters
9. Professional and Responsive Dashboard Design

# Key Insights

1. Identifies top-performing product categories.
2. Compares revenue across outlet sizes and locations.
3. Highlights sales trends based on outlet establishment year.
4. Evaluates customer satisfaction using average ratings.
5. Provides a geographical overview of sales distribution across India.
